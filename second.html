Question-3: What is event delegation in the DOM?
When users interact with dom like click hover then dom fired and JavaScript find the event where occurred, from the window to the target step by step this process called event delegation.
Question-4: What is event bubbling in the DOM?
Event bubbling is when a event occurred in in an element its also occurred in parent element because the parent event handler also response when the child received an event thats called event bubbling.
